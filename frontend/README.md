# Personalized Learning Analytics & Study Time Analyzer

##  Overview
Personalized Learning Analytics & Study Time Analyzer is an AI-powered web application that analyzes students' uploaded semester marksheets to evaluate academic performance, 
identify weak subjects, and provide personalized learning recommendations. 
The platform also includes a Study Time Analyzer that helps students monitor study patterns and improve learning efficiency.



##  Features
- Upload and analyze semester marksheets
- Identify weak and failed subjects
- AI-powered personalized learning recommendations
- Study Time Analyzer for tracking study hours
- Performance analysis dashboard
- Interactive and responsive user interface
